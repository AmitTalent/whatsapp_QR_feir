# whatsapp_QR_feir
קוד QR לכנס משתחררים שמפנה לווצאפ של רכז גיוס.
כיום הקישור מכוון לרכזת גיוס שלנו, ולכנס משתחררים באשקלון.
כל פעם שהקישור יכוון למשרות אחרות - נוסיף ונשנה את המשרות והמיקומים ונעדכן תאיך של השינוי, כמו גם נעדכן את ההודעה כך שתדבר על הכנס הנוכחי.

עדכון אחרון 29/01 16:20

עדכון אחרון 03/02 10:00 - שינוי הלוגיקה כך שתהיה נפרדת מהכפתור:
- מפשט את הקוד. 
- ניתן לשנות את שם הכנס בקלות.
- ניתן להוסיף מספרי טלפון של רכזים לאזורים שונים בארץ.

עדכון אחרון 03/02 14:20 - הוספת ערכים במערך:
- ניתן לשנות או להוסיף ערך של אזור יותר בקלות ביחד עם מספר טלפון.
- ניתן לשנות או להוסיף משרה חדשה בקלות מתוך הרשימה בלי לשנות את הקוד עצמו.
